# ROPRQUEST

## Install

```bash
$ bundle install
```

## Begin ROPRing

```bash
rake
```

## Testing

```bash
rake test
```

## Bumping the version

Edit the version number in `lib/roprquest/version.rb` then run:

```bash
bundle install
```

The `Gemfile.lock` file should be checked in.
